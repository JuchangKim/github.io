Phone Virus Simulation

Download Phone Virus Simulation

Using Knowledge

Thread, Synchronisation, Generics, GUI, Key input, Lists, etc.

Explanation

There is a simulation of virus-infected phones. The blue dot is a healthy phone; the red dot is an infected phone. Infected phones must go to a repair shop at the middle top of the panel. The repair shop can only fix one phone at a time, so other infected phones must wait.

After a certain time, the repaired phone is released as a healthy phone, and the repair shop takes in another waiting infected phone. When you press the up arrow key, a new healthy phone is created. Press the v key to infect a healthy phone randomly. If a healthy phone is near an infected phone, it also becomes infected.

Path Finder (Binary Maze)

Download Binary Maze

  • Maze Text File 1
  • Maze Text File 2

Using Knowledge

Binary Tree, GUI, Text File Input, Lists, Traversal with Stack, Recursion, etc.

Explanation

The program first requests a maze text file. It reads the text and uses the data to build a binary-tree-based maze. The maze is displayed on screen, and a traversal algorithm (using recursion and stack) tries to find the exit. Once the exit is found, the path is shown in the bottom-right area.
