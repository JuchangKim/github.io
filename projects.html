Algorithm Research: Currency Exchange (Python)


Overview

This research explores currency exchange rates and finding the most efficient conversion paths using classic graph algorithms such as Bellman-Ford and Floyd-Warshall. The problem is modeled as a directed graph, where each edge represents a potential currency exchange rate from one currency to another.


Research Paper (PDF)



Download PDF

Python Code

Download currency_exchange.zip

Key Algorithms & Concepts

    Directed Graph Representation: Each currency is a node, each exchange rate is a directed edge.


    Bellman-Ford Algorithm: Used to handle negative edge weights (e.g., arbitrage opportunities) and detect negative cycles.


    Floyd-Warshall Algorithm: Computes all-pairs shortest paths, evaluating every possible path among currencies.


Reflection

Through this research, I gained experience in implementing and comparing Bellman-Ford and Floyd-Warshall on a real-world currency exchange scenario. The project solidified my understanding of directed graph algorithms, handling negative edge weights, and how these algorithms can be applied to financial problems such as arbitrage detection.
